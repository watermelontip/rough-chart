# ✏️ 手绘风格图表生成器

> 一个使用 Canvas API 绘制手绘/素描风格图表的在线工具，纯前端实现，零依赖。

## 🌐 在线演示

**👉 [点击体验](https://watermelontip.github.io/rough-chart/)**

## ✨ 功能特性

- 📊 **柱状图** - 手绘风格柱状图展示
- 📈 **折线图** - 素描风格折线趋势图
- 🥧 **饼图** - 手绘饼图比例展示
- 🍩 **环形图** - 环形图数据可视化
- ⚬ **散点图** - 手绘散点分布图
- 🎨 **9种预设配色方案** - 糖果、海洋、森林、夕阳等
- 🖌️ **自定义颜色** - 自由选择配色
- 📝 **自定义数据** - 文本框输入标签和数值
- 📋 **数据预设** - 内置月度销售额、气温、成绩等示例
- ⚙️ **图表设置** - 标题、轴标签、网格、手绘粗细可调
- 📥 **导出PNG** - 一键导出高清图片
- 📥 **导出SVG** - 矢量图导出
- 🎲 **随机抖动** - 每次渲染不同的手绘效果
- 🌙 **深色主题** - 现代暗色 UI 设计
- 📱 **响应式布局** - 适配各种屏幕尺寸
- 🇨🇳 **中文界面** - 全中文操作界面

## 🛠️ 技术实现

- **纯 HTML/CSS/JavaScript** - 零外部依赖
- **Canvas 2D API** - 手绘效果核心引擎
- **手绘算法** - 通过贝塞尔曲线 + 随机抖动模拟手绘笔触
- **双重描边** - 每条路径绘制两次以增强素描感
- **Seeded Random** - 可重复的手绘抖动效果

## 📦 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需任何构建步骤或服务器。

1. 选择图表类型
2. 输入数据或选择预设
3. 选择配色方案
4. 调整图表设置
5. 导出为 PNG 或 SVG
